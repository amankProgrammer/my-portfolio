# 🚀 Cinematic 3D Portfolio

A highly immersive, cinematic, and interactive developer portfolio built with React 19, Tailwind CSS 4, and Framer Motion. Featuring an AI Assistant powered by Google Gemini and ultra-realistic 3D tilt effects.

---

## ✨ Key Features

- **🤖 Gemini-Powered AI Assistant**: A smart chatbot that knows everything about my journey, skills, and projects.
- **🌌 Cinematic 3D Experience**: Ultra-realistic 3D tilt effects and smooth floating animations using Framer Motion.
- **📱 Responsive Design**: Fully optimized for all screen sizes, from mobile to ultra-wide displays.
- **📅 Interactive Timeline**: A visual journey of my career and educational milestones.
- **🛠️ Tech Stack Showcase**: Beautifully animated tech cards for a modern look.
- **📧 Dynamic Contact Form**: Integrated contact section for seamless communication.

---

## 🛠️ Tech Stack

### Frontend & Styling
- **React 19**: Latest features and performance optimizations.
- **Vite**: Ultra-fast next-generation frontend tooling.
- **Tailwind CSS 4**: Modern utility-first CSS framework for rapid UI development.
- **Framer Motion**: Powering the cinematic animations and 3D effects.
- **Lucide React**: Clean and consistent icon set.

### AI & Backend Services
- **Google Gemini 1.5 Flash**: Powering the intelligent portfolio assistant.
- **Firebase**: Backend services for deployment and potentially more.

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amankProgrammer/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
